1) El sistema de control de versión Git lo inventó Linus Torvalds para el desarrollo de Linux kernel. Se vió en la necesidad de crearlo ya que al programa que usaban previamente (BitKeeper) se le retiró el uso gratis. El dueño de los derechos, Larry McVoy, adució que un competidor habia aprovechado su codigo para crear SourcePuller.
